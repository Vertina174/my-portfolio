# 🌐 Personal Portfolio Website

This is my personal portfolio website built to showcase my projects, skills, and development work.

🔗 Live Site: https://my-portfolio-nisw.vercel.app/

---

## 🚀 Tech Stack

- **Astro** – used for building a fast, lightweight website  
- **Vercel** – deployment and hosting  
- Minimal JavaScript usage for better performance  

---

## ⚡ Why Astro?

I chose Astro because:
- It is lightweight and optimized for performance  
- Uses minimal JavaScript by default  
- Faster load times compared to traditional frameworks
